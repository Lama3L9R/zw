## 朱在群中曰：「明天大家尽量早点交作业，课前交我给你现判八成耽误上课」$^{①}$
## 驼$^{②}$私问：「为啥课前交只判八成？」
## 朱回问：「？你母语是汉语吗」

::: tip 解释
尊敬的朱老师在群中说：「明天大家尽量早点交作业，课前交我给你现判八成党务上课」
驼在私下问道：「为啥可签交只判八成？」
尊敬的朱老师回复我并问道：「？你母语是汉语吗」
:::

::: tip 注解
- 这句话的正确断句为：明天大家尽量早点交作业，课前交我 **/** **(我)** 给你现判 **/** 八成耽误上课
- **驼**：指站长 `@lamadaemon`
:::
